Descrição do Produto

1. Sistema de suporte: 10 e Windows 7 (o software V-Volvo Hino Perkins Cat Nissan deve ser usado no sistema xp, ou outro software pode ser usado no Windows 10 64 ou 7 32bit).

2. Compatível com aplicativos que diagnosticam motores, transmissões, ABS, painéis de instrumentos e muito mais.
3. Reduza o tempo de reprogramação com conexão USB direta.
4. Recursos de frota mista com amplo suporte a protocolo.
5. Acesso rápido aos dados do veículo para relatórios de manutenção ou gerenciamento.
6. Conecta-se rapidamente ao conector de diagnóstico do veículo.
7. Fácil portabilidade aumenta a flexibilidade. Pode ser usado com PC ou laptop.

Aviso prévio:
1. Não pode ser atualizado; caso contrário, o multiplexador será danificado.
2. Para os softwares Vol-vo, Hino, Cat e Ni-ssan, instale-os apenas no Windows XP, outro software de veículo pode ser instalado no Win 7.
3. Quando você altera a hora do seu laptop, o software Cu-mmins Inline fica bloqueado, faça o download do arquivo anexado ow-to-unlock-cum-mins-inline "para resolvê-lo.
4. Antes de instalar o CD do software Nexiq, reinstale o sistema do computador primeiro, caso contrário, o multiplexador será danificado.
