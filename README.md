# Voltrucks 🚛⚡

Site institucional e catálogo de produtos da **Voltrucks**, focado em soluções profissionais para diagnóstico automotivo pesado, scanners, softwares técnicos, catálogos de peças e ferramentas para caminhões, ônibus, máquinas agrícolas e equipamentos industriais.

---

## 📌 Sobre o Projeto

O projeto Voltrucks foi desenvolvido para apresentar produtos técnicos de forma clara, organizada e escalável, permitindo:

- Exibição de múltiplos produtos
- Páginas individuais para cada produto
- Descrições longas e técnicas
- Integração com WhatsApp para vendas
- Carrinho de compras com JavaScript
- Estrutura simples em HTML, CSS e JS puro

---

## 🧩 Padrões Utilizados

- HTML semântico (`section`, `header`, `footer`, `main`)
- Uma página HTML por produto
- Descrição do produto dentro de uma única `<div class="produto-descricao">`
- Uso mínimo de tags (`p`, `br`, `h2`, `ul`, `li`)
- Estilo controlado apenas por CSS
- JavaScript separado por responsabilidade

---

## 🛒 Carrinho de Compras

- Armazenamento via `localStorage`
- Suporte a múltiplos produtos
- Contador no ícone do carrinho
- Finalização via WhatsApp

---

## 💬 Contato / Vendas

A finalização de pedidos é feita diretamente via **WhatsApp**, facilitando o atendimento personalizado e o envio rápido das informações do pedido.

---

## ⚙️ Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript (Vanilla)
- Bootstrap (quando necessário)

---

## 📄 Observações

- Projeto leve e fácil de manter
- Ideal para catálogos técnicos com descrições longas
- Estrutura preparada para crescimento futuro
- Pode ser facilmente integrado a backend ou CMS
